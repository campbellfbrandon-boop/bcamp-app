Phase 1 Coach's Note

The BCAMP hybrid keeps metabolic demand high through supersets on all isolation work and HIIT finishers — critical for burning fat while building muscle. Glute specialization (cable kickbacks + 2nd hip thrust session) starts Week 3. Deload every 4th week — non-negotiable. Log every set; progressive overload is the engine.

Progression & Recovery

Double Progression

Hit the top of the rep range for ALL sets before adding weight. Hit 4×12 when the target is 4×8–12? Add 2.5–5kg next session. Every set logged in this app is your evidence.

Cardio Protocol

3× HIIT (10–12 min, 30s on / 30s off) attached to non-leg days. 1× LISS walk (25–35 min) on rest day. If fat loss stalls for 2+ weeks: add one 20-min LISS session before adjusting calories.

Deload Weeks (4, 8, 12, 16)

50% volume, 55–65% of working weight, compounds only, zero HIIT, zero isolation. Eat at maintenance with full carb refeed. Sleep 9h. This is when adaptation happens.

Recovery

Sleep

8–9h non-negotiable. GH peaks in slow-wave sleep. Cool room, no screens 1h before. Highest-ROI recovery tool.

Stress

Chronic cortisol = fat storage + catabolism. Walk, breathe, manage workload. You adapt at rest — not during training.

Mobility

5 min foam roll post-session. Hip flexors + thoracic daily. These minutes protect your years of investment.

Phase 2 Coach's Note

The split changes here because each muscle hit twice weekly grows ~50% faster for advanced trainees. The HIIT finishers reduce — calories are needed for muscle growth. The OHP 5×5 and Deadlift/Bench strength blocks are the shoulder and back width builders. Saturday's 6 sets of hip thrusts is the primary glute mass session of the week.

Training Days
Progression & Bulk Notes

Strength Blocks (Weeks 21, 29–30)

On designated strength weeks, lower the rep range to 3–5 on all primary compounds and add 5–10 lbs. Strength gains on a bulk are fast — ride them. New PRs in this phase set the ceiling for Phase 3.

Weekly Weight Gain Check

Weigh daily, track the 7-day average. Target: 0.25–0.5 lbs/week. Gaining faster? Cut surplus by 100 cal. Slower? Add 100 cal. Don't react to single-day fluctuations — only weekly averages.

Deload Weeks (20, 24, 28, 32)

Same deload protocol — 50% volume, compounds only, eat at maintenance. On bulks, deload weeks allow CNS recovery and often result in large strength jumps the following week. Trust the process.

Recovery

Sleep

On a bulk, 8–9h sleep maximizes GH release and muscle protein synthesis. Don't skimp — you're trying to grow.

Cardio

Reduced to 2× HIIT + 1–2× LISS on a bulk. The surplus calories are for muscle — excessive cardio fights the surplus.

Mobility

Heavier loads in Phase 2 make mobility more important, not less. Hip flexors and thoracic spine daily. Protect the joints.

Phase 3 Specialization Blocks

Weeks 39–41: Glute Peak

Hip thrust 6×10 on BOTH sessions. RDL 4×15 both sessions. Cable kickback 4×20/leg. This is the highest glute volume of the entire year — and it shows.

Weeks 43–44: Back Thickness

Add rack pulls. Chest-supported DB row. Three pulling movements per session. Back frequency bumped to 3×/week. These two weeks add more back thickness than a month of regular training.

Weeks 45–46: Shoulder Width

Bench, Squat, Deadlift, OHP, Pull-Up — formal 1RM test. Compare every number to your Week 1 baseline. This is your Year 1 result. Log it here in the app.

Meal Timing

Training Day

  • Carb-rich meal 1–2h pre-workout (rice, oats, banana + protein)
  • Post-workout within 60 min: whey + fast carbs (fruit, white rice)
  • Whole food proteins: chicken, beef, eggs, fish — prioritize these
  • Fiber: 25–35g/day from vegetables and legumes
  • Creatine: 3–5g daily — every day regardless of training

Rest Day

  • Reduce carbs 20–30% — fat oxidation runs higher at rest
  • Keep protein identical — MPS runs 24–48h post-training
  • Increase fibrous vegetables for satiety without excess calories
  • Magnesium glycinate 200–400mg before bed for sleep quality
  • Minimum 3–4L water on training days

APPLE WATCH
Web apps can't directly control Apple Watch — that requires a native App Store app built in Swift. The best workaround is a one-tap Shortcut on your home screen that starts a Strength Training session on your Watch instantly.
Create a 1-Tap Watch Shortcut
  1. Open the Shortcuts app on your iPhone
  2. Tap + to create a new shortcut
  3. Tap Add Action → search "Log Workout"
  4. Set Workout Type to Strength Training
  5. Tap the shortcut name at the top → rename it "Start BCAMP"
  6. Tap Done
  7. Long-press the shortcut → Add to Home Screen
⚡ Now you have two icons side by side on your home screen — BCAMP for logging, and "Start BCAMP" to kick off the Watch workout. One tap each.

GOOGLE SHEETS SYNC
Every time you save a set log, BCAMP will write it to your Google Sheet. Re-saving the same exercise on the same day overwrites the existing row — no duplicates.
Step 1 — Create the Apps Script (one time only)
  1. Open script.google.com and click New Project
  2. Delete any existing code and paste the script below
  3. Click Deploy → New Deployment → Web App
  4. Set Execute as: Me  ·  Who has access: Anyone
  5. Authorize when prompted, then copy the Web App URL
function doPost(e) {
  try {
    var data = JSON.parse(e.postData.contents);
    var ss   = SpreadsheetApp.getActiveSpreadsheet();

    // Per-exercise tab
    var sheetName = data.exercise.substring(0, 50);
    var sheet = ss.getSheetByName(sheetName);
    if (!sheet) {
      sheet = ss.insertSheet(sheetName);
      sheet.appendRow(['Date','Set','Weight (lbs)','Reps','Completed','Est 1RM (lbs)','Phase','Day']);
      sheet.getRange(1,1,1,8).setFontWeight('bold').setBackground('#1c2b3a').setFontColor('#ffffff');
      sheet.setFrozenRows(1);
    }

    // Delete same-day rows (overwrite behaviour)
    var lastRow = sheet.getLastRow();
    if (lastRow > 1) {
      var rows = sheet.getRange(2,1,lastRow-1,1).getValues();
      for (var i = rows.length-1; i >= 0; i--) {
        if (rows[i][0] === data.date) sheet.deleteRow(i + 2);
      }
    }

    // Append new rows
    data.sets.forEach(function(s) {
      var w = parseFloat(s.weight)||0, r = parseInt(s.reps)||0;
      var e1rm = (w && r) ? Math.round(w*(1+r/30)) : '';
      sheet.appendRow([data.date, s.set, s.weight, s.reps, s.done?'Yes':'No', e1rm, data.phase||'', data.day||'']);
    });

    // Master log tab
    var master = ss.getSheetByName('_Master Log');
    if (!master) {
      master = ss.insertSheet('_Master Log');
      master.appendRow(['Date','Exercise','Set','Weight (lbs)','Reps','Completed','Est 1RM','Phase','Day']);
      master.getRange(1,1,1,9).setFontWeight('bold').setBackground('#1c2b3a').setFontColor('#ffffff');
      master.setFrozenRows(1);
    }
    var mLast = master.getLastRow();
    if (mLast > 1) {
      var mData = master.getRange(2,1,mLast-1,2).getValues();
      for (var j = mData.length-1; j >= 0; j--) {
        if (mData[j][0]===data.date && mData[j][1]===sheetName) master.deleteRow(j+2);
      }
    }
    data.sets.forEach(function(s) {
      var w = parseFloat(s.weight)||0, r = parseInt(s.reps)||0;
      var e1rm = (w&&r) ? Math.round(w*(1+r/30)) : '';
      master.appendRow([data.date, sheetName, s.set, s.weight, s.reps, s.done?'Yes':'No', e1rm, data.phase||'', data.day||'']);
    });

    return ContentService.createTextOutput(JSON.stringify({status:'ok'}))
      .setMimeType(ContentService.MimeType.JSON);
  } catch(err) {
    return ContentService.createTextOutput(JSON.stringify({status:'error',message:err.toString()}))
      .setMimeType(ContentService.MimeType.JSON);
  }
}

RESTORE FROM SHEETS
Pulls your full workout history from Google Sheets back into this device. This will merge with any existing local data — existing entries for the same exercise on the same date will be overwritten by the Sheet version.
Required: Add doGet to your Apps Script
Your Apps Script needs a doGet handler so the app can read data back. Add this function to your existing script, then create a new deployment.
function doGet(e) {
  try {
    var ss     = getOrCreateSpreadsheet();
    var master = ss.getSheetByName('_Master Log');
    if (!master || master.getLastRow() < 2)
      return ContentService.createTextOutput(JSON.stringify({ status:'ok', rows:[] }))
        .setMimeType(ContentService.MimeType.JSON);

    var data  = master.getDataRange().getValues();
    var headers = data[0]; // Date,Exercise,Set,Weight,Reps,Completed,Est1RM,Phase,Day
    var rows  = [];
    for (var i = 1; i < data.length; i++) {
      var r = data[i];
      rows.push({
        date:      String(r[0] instanceof Date ? Utilities.formatDate(r[0], Session.getScriptTimeZone(), 'yyyy-MM-dd') : r[0]),
        exercise:  String(r[1]),
        set:       Number(r[2]),
        weight:    String(r[3]),
        reps:      String(r[4]),
        completed: String(r[5]),
        phase:     String(r[7] || ''),
        day:       String(r[8] || '')
      });
    }
    return ContentService.createTextOutput(JSON.stringify({ status:'ok', rows:rows }))
      .setMimeType(ContentService.MimeType.JSON);
  } catch(err) {
    return ContentService.createTextOutput(JSON.stringify({ status:'error', message:err.toString() }))
      .setMimeType(ContentService.MimeType.JSON);
  }
}
